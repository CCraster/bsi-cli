## Install

```
npm install -g bsi-cli
```

## Usage

### Create

create framework template for frontend,including

- vue admin
- vue component
- react
- express
- koa

```
bsi create <template-name>
```

### Date

transfrom timestamps to formated(yyyy-MM-dd HH:mm:ss) value, vice versa

```
bsi date <value>
```
